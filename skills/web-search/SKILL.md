# 网页搜索技能

## 什么时候用
当用户需要查找最新信息、新闻、或实时数据时使用搜索工具。

## 搜索策略
1. **关键词提取**：从用户问题中提取核心关键词
2. **搜索查询**：用关键词构造搜索查询
3. **结果整理**：把搜索结果整理成清晰的回答

## 注意事项
- 搜索结果是模拟的，实际项目中对接真实搜索 API
- 回答时注明信息来源
- 如果搜索结果不够，尝试换关键词再搜一次

## 回答格式
搜索结果应包含：
- 直接回答用户的问题
- 信息来源
- 相关补充（如有）
